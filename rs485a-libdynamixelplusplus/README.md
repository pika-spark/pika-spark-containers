:sparkles: `rs485a-libdynamixelplusplus`
========================================
This Docker container allows to ping all Dynamixel servos connected to RS485A / `/dev/ttymxc1`. 

<p align="center">
  <a href="https://pika-spark.io/"><img src="https://raw.githubusercontent.com/pika-spark/.github/main/logo/logo-pika-spark-bg-white-github.png" width="40%"></a>
</p>

### How-to-build/run
```bash
git clone https://github.com/pika-spark/pika-spark-containers
cd pika-spark-containers/rs485a-libdynamixelplusplus
./docker-build.sh
sudo ./docker-run.sh
```
