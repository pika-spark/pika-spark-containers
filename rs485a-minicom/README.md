:sparkles: `rs485a-minicom`
===========================
This Docker container allows to send/receive data via RS485A via `minicom`. 

<p align="center">
  <a href="https://pika-spark.io/"><img src="https://raw.githubusercontent.com/pika-spark/.github/main/logo/logo-pika-spark-bg-white-github.png" width="40%"></a>
</p>

### How-to-build/run
```bash
git clone https://github.com/pika-spark/pika-spark-containers
cd pika-spark-containers/rs485a-minicom
./docker-build.sh
sudo ./docker-run.sh [baudrate]
```
